Jinja Templating Techniques

for loops with Python lists

{% for o in output %}

{{o}}

{% endfor %}

if statements and |int and |string keywords

{% if output[4]|int > 10 %}

42 is greater than 10.

{% endif %}

|replace() methods

{{ output[0]|replace('good','AMAZING!!') }}

for loop through string by splitting to list

    {% for lang in output[1].split(',') %}
  • {{lang}}
    • {% endfor %}

Render HTML in |safe mode

{{output[2]|safe}}

Render "raw" code with <pre> tags

{% raw %} 
    {% for i in items %}
      {{item}}
    {% endfor %}
{% endraw %}

Macros for easy inputs and repetitive tags

{% macro input(name, value='', type='text', size=20) -%} {%- endmacro %} The macro can then be called like a function in the namespace:

{{ input('username') }}

{{ input('password', type='password') }}
